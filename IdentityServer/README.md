# 🔐 IdentityServer

**IdentityServer** — это инфраструктурный модуль (Integration Layer), обеспечивающий безопасную и гибкую аутентификацию пользователей в экосистеме приложения.

Проект выступает в роли адаптера, который инкапсулирует работу с `Microsoft.AspNetCore.Identity`, реализуя внутренние интерфейсы-порты ядра (например, `ISimpleAuthenticationPort`). Это позволяет полностью изолировать бизнес-логику системы от деталей реализации механизмов авторизации.

## ✨ Ключевые возможности

* 🛡 **Гибкие схемы аутентификации:** Динамическое переключение между `IdentityConstants.ApplicationScheme` (Cookies) и `IdentityConstants.BearerScheme` (JWT) в зависимости от типа клиента.
* 🔄 **Управление жизненным циклом сессий:** Бесшовная пролонгация сессий (`RefreshSignIn`), безопасный выход из системы и управление блокировками аккаунтов (lockout).
* 🧩 **Бесшовная интеграция с Security-контекстом:** Автоматическая синхронизация аутентифицированного `Principal` с кастомным интерфейсом `ISecurityContext` приложения.
