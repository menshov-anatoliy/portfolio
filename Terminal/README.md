# 📟 Terminal — складские терминалы

**Терминалы** — корпоративная подсистема управления складскими операциями, обеспечивающая эффективное взаимодействие между складскими работниками и бизнес-логикой. Система построена на **ASP.NET Core 9** с архитектурой, ориентированной на чистый код и расширяемость.

---

## 🧩 Компоненты системы

| Компонент | Технология |
|---|---|
| 🌐 **API** | Универсальный REST API на ASP.NET Core 9 |
| 📱 **Web-клиент** | Angular SPA |
| 🖥️ **Windows-клиент** | WinForms |

---

## ✨ Ключевые возможности

- 🌐 **Единая точка обработки запросов** — универсальный эндпоинт `POST api/terminal/postResponse` для всех клиентских сценариев.
- 🔐 **Аутентификация и авторизация** — через `IdentityServer` с поддержкой **cookie** и **bearer**-схем.
- 🧵 **Цепочка middleware** — многоуровневая обработка для управления сессиями и контекстом безопасности.
- 📖 **Документация API** — встроенный Swagger / OpenAPI «из коробки».
- 🚀 **Интеграция с подсистемой деплоя** — поставка бизнес-логики под конкретных клиентов.
- 💾 **Подсистема кэширования** — прозрачное ускорение часто запрашиваемых данных.
