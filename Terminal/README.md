# Складские терминалы

**Терминалы** — корпоративная система для управления складскими операциями, обеспечивающая эффективное взаимодействие между складскими работниками и бизнес-логикой. Система построена на ASP.NET Core 9 с архитектурой, ориентированной на чистый код и расширяемость.

## ⚙️ Компоненты системы

- Универсальный REST API (ASP.NET Core 9)
- Web-клиент (Angular SPA)
- Windows-клиент (WinForms)

## ✨ Ключевые возможности

- Единая точка обработки запросов через универсальный API (`POST api/terminal/postResponse`)
- Аутентификация и авторизация через IdentityServer с поддержкой cookie и bearer-схем
- Многоуровневая цепочка middleware для управления сессиями и контекстом безопасности
- Встроенная документация API через Swagger / OpenAPI
- Интеграция с подсистемой деплоя бизнес-логики под конкретных клиентов
- Подсистема кэширования данных
