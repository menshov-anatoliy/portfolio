# Portfolio

## Презентационная часть

Этот репозиторий — портфолио с примерами аккуратного и структурированного подхода к работе с кодом и документацией.  
Цель — показать, как собирать, систематизировать и описывать исходные материалы в понятном человеку формате.

## Описание файлов

- `tools/collect_org_examples.py`  
  Скрипт, который получает репозитории организации `OOO-IT-MOLCOM`, выбирает показательные исходные файлы и формирует единый документ с примерами.

- `examples/ooo-it-molcom-code-examples.md`  
  Итоговый файл с собранными примерами кода, ссылками на источники и краткими описаниями.
