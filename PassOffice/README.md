# 🏢 PassOffice — бюро пропусков

**Бюро пропусков** — комплексное решение для автоматизации процессов бюро пропусков и эффективного контроля доступа на территорию складского комплекса. Система охватывает **полный жизненный цикл пропуска** — от подачи заявки до автоматизированного выезда через КПП.

> 🎯 Проект нацелен на минимизацию очередей в бюро пропусков, исключение человеческого фактора при проверке разрешений и обеспечение полной прозрачности перемещений на территории.

---

## ✨ Ключевые возможности

- 🎫 **Гибкая система заявок** — поддержка различных типов пропусков (разовые на 20 часов, временные на период договора, постоянные) для водителей, пешеходов, гостей и арендаторов.
- 🔗 **Удалённое заполнение** — отправка ссылки внешним пользователям для самостоятельного дозаполнения данных, что сокращает нагрузку на персонал.
- 🚧 **Автоматизированные КПП** — интеграция с камерами для распознавания номеров ТС, автоматическое определение наличия пропуска и управление шлагбаумами.
- ✅ **Многоуровневое согласование** — настраиваемые маршруты одобрения заявок с автоматическими уведомлениями ответственных лиц.
- 🧾 **Контроль убытия** — обязательные отметки от склада или арендатора о завершении работ, без которых выезд через КПП может быть заблокирован.

---

## ⚙️ Технологические особенности

- 📷 **Фотофиксация** — поддержка фотографирования посетителей непосредственно в интерфейсе системы.
- 🔁 **Реверсивное движение** — программная настройка стороны движения для реверсивных полос на КПП.
- ⏱️ **Мониторинг в реальном времени** — автоматическая фиксация времени прибытия и убытия на основе данных со шлагбаумов.
